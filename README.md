# Rank and File

> Bringing fantasy rankings to the masses. **Rank and File** is a social platform for groups/leagues to compare rankings and accuracy.

## Design Language
**Modern, Premium, Glassmorphic, Delightful, Thoughtful.**

We aim for a high-end feel with:
- **Glassmorphism**: Translucent layers, blurs, and depth.
- **Premium**: Refined typography, generous whitespace, and rich colors.
- **Delightful**: Smooth animations and micro-interactions.
